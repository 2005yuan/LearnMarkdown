### 无序列表

## 5.代码格式

### 单个代码行

一个代码行可以用两个反引号`框起来

`这是代码`

### 代码区块

可以用\`\`\`包裹代码行，反引号旁边可以写代码语言种类
```C++
void swap(int v[], int i, int j) {
	int temp = v[i];
	v[i] = v[j];
	v[j] = temp;
}

void qsort(int v[], int left, int right) { //升序排序
	if (right <= left)
		return ;
	swap(v, left, (left + right) / 2);
	int last = left;
	for (int i = left + 1; i <= right; i++)
		if (v[left] > v[i]) //大小判断
			swap(v, ++last, i);
	swap(v, last, left);
	qsort(v, last + 1, right);
	qsort(v, left, last - 1);
}
```

## 9.高级技巧

### **markdown支持html元素**

目前支持的元素有：\<br\>，\<kbd\>，\<b\>，\<i\>，\<em\>，\<sup\>，\<sub\>等

####  \<br\>

这是个换行符

#### \<kbd\>

表示文本是从键盘上键入的，要和</kbd>一起使用

例如：使用 <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>Del</kbd> 重启电脑

#### \<b\>

表示加粗文本，要和\</b\>一起使用

例如：<b>加粗</b>

#### \<i\>

表示斜体文本，要和\</i\>一起使用

例如：<i>斜体</i>

#### <em\>

表示强调文本，要和\</em\>一起使用

例如：<em>强调</em>

#### \<sup\>

表示上标，要和\</sup\>一起使用

例如：其他字<sup>上标</sup>

#### \<sub\>

表示下标，要和\</sub\>一起使用

例如：其他字<sub>下标</sub>

### 转义语法

许多字符都有特定含义，要显示特定字符则需要转义字符\

通过\\\ +  特定字符 的方式实现

### 公式

\$\$...\$\$ 或者 \\\[...\\\] 或者 \`\`\`math 中的数学表达式将会在块内显示。

其中\\\\表示换行

例如：
$$
f(x) = sinx + 32 \\
g(x) = f(x + 2) + f(x - 2)
$$
